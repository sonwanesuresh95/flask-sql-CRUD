# flask-sql-CRUD
Simple Webapp with sqlite3 db for CRUD operations
## Info
Webapp for CRUD operations on records with SQL database:<br>
1. Create records
2. Read records
3. Update records
4. Delete records

## Usage
```
$ pip install requirements.txt
$ python app.py
```
